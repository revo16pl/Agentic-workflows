# editorial_review.md

## How to use
- Czytaj draft jak redaktor, nie jak automat QA.
- Jeśli zdanie brzmi nienaturalnie po przeczytaniu na głos, wpisz je tutaj i przepisz od zera.
- Nie poprawiaj tylko pojedynczych słów. Przepisuj całe fragmenty, jeśli sens lub rytm są słabe.
- Zapisz minimum 2 iteracje przed final approval.
- Po machine QA zrób jeszcze jeden krótki pass i wdroż ewentualne poprawki, zanim ustawisz final approval.
